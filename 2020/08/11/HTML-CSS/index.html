Position

position 속성: static, absolute, relative, fixed, sticky 5가지 값을 가질 수 있다.
postion 속성의 기본값으로 static을 채택한다.

값 의미 기본값
static top, bottom, right, left 값을 가질 수 없는 normal flow이다. static
absolute 부모의 position 값을 기준으로 콘텐츠가 위치하는 방식
relativ 자신을 기준으로 좌표를 지정하는 방식
fixed 뷰표트를 기준으로 좌표를 가지며, 따로 스크롤이 되지 않고 뷰포트에 고정되는 방식
